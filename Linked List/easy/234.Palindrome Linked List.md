```diff
## Solution :  TC: O(n) / SC: O(n)
```                           
```python
class Solution:
    def isPalindrome(self, head: Optional[ListNode]) -> bool:
        ans=[]
        while head:
            ans.append(head.val)
            head=head.next
        return ans==ans[::-1]
```
