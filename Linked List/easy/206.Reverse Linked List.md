```diff
## Solution :  TC: O(m+n) / SC: O(1)
```                           
```python
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        last=None
        while head: last,head.next,head=head,last,head.next
        return last
```
