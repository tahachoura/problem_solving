```diff
## Solution :
```                           
```python
class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        a=headA ; b=headB
        while a!=b:
            a=headB if not a else a.next
            b=headA if not b else b.next
        return a
```
